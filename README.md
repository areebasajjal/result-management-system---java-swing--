# Student Result Management System (Java OOP + Swing GUI)

## Overview
A Java-based desktop application for managing students, faculty, courses, and academic results using OOP principles and a Swing GUI. The system also supports file-based data persistence.

---

## Features
- Student management (Arts, Science, Engineering)
- Course management
- Faculty management
- Result entry and calculation
- Transcript generation
- Persistent data storage using serialization
- GUI-based interface using Java Swing

---

## Tech Stack
- Java
- OOP Principles
- Swing (GUI)
- File Handling / Serialization

---

## Project Structure
- backend/ → Core logic and models
- gui/ → User interface screens
- resources/ → Images and assets
- DataStore → Handles file persistence
- Uml → class diagram 

---

## How to Run
1. Open project in any Java IDE (IntelliJ / Eclipse / NetBeans / vs code)
2. Run `mainPage.java`
3. Ensure project folders remain unchanged

---

## Purpose
This system was developed as a second-semester Object-Oriented Programming project to demonstrate understanding of Java OOP and GUI development.
